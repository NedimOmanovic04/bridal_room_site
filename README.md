## Live Demo
[bridalroomvisoko.pages.dev](https://bridalroomvisoko.pages.dev)

## Features
- Luxury bridal dress showcase with filters (category, price range, availability)
- Admin panel with authentication for managing dresses and categories
- Image upload to Supabase Storage
- Contact form with email delivery via Resend API
- Smooth animations with Framer Motion
- Fully responsive design optimized for mobile devices

## Tech Stack
- Next.js 15 (App Router, Edge Runtime)
- TypeScript
- Tailwind CSS v4
- Supabase (PostgreSQL + Storage)
- Resend API
- Framer Motion
- Cloudflare Pages (hosting)

## What I Learned
- Building a full-stack Next.js application using the App Router architecture
- Integrating Supabase for database management and image storage
- Sending transactional emails via the Resend API
- Deploying a Next.js app to Cloudflare Pages with Edge Runtime
- Implementing an admin panel with authentication without third-party auth services
- Dynamic product filtering and image optimization
